# **App Name**: Family Firm Focus

## Core Features:

- Curriculum Display: Display a scrolling view of all available modules and lessons on the homepage.
- Content Summary: Show a quick preview or summary for each module
- External Resource Linking: Provide links to external posts, blogs, playlists, books, or other educational resources related to each module.
- Progress Tracking: Allow users to mark content as 'complete' to track learning progress. When clicked, store it to the user profile

## Style Guidelines:

- Primary color: Dark blue (#34495E), conveys stability and professionalism, which are crucial in business contexts.
- Background color: Light gray (#F2F3F4), offers a neutral backdrop that ensures readability and reduces eye strain during long study sessions.
- Accent color: Warm orange (#E67E22), is used to highlight interactive elements like links and the progress checkbox, drawing user attention without overwhelming the design.
- Body and headline font: 'Inter', a grotesque-style sans-serif chosen for its clean appearance that facilitates easy reading on digital interfaces, is appropriate for both headings and body text.
- Use simple, professional icons to represent different modules and resource types.
- Use a card-based layout to organize content. Make the cards responsive for various screen sizes.
- Use subtle transitions when opening content pages and marking lessons as complete.
